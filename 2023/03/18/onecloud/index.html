前言

玩客云一台老矿渣，目前 pdd 32 左右大洋能买到

但是本身玩客云存储空间仅 8g,所以还是买了根易驱线

要刷机还得整根公对公的 A 口 USB 线

总共花费软妹币 43，因为我 12v-1.5a dc 电源线以及硬盘都是有的

需求:

  • 离线下载 aria
  • 内网穿透 tailscale/phddns
  • 网盘管理 alist

为了环境不混乱，我选择 docker 的形式

刷机

软件工具:

  • USB_Burn
  • 直刷包
  • SSH 工具-推荐MobaXterm

刷机过程:

  1. 安装 usb_burn，ssh 工具,下载直刷包
  2. 打开 usb_burn 导入直刷包

    选择刚刚下载的直刷包,点击打开
  3. USB 线连接电脑 usb 口以及玩客云靠近 hdmi 的那个 usb 口

    刷过 armbian 的按住 reset 然后插入电源即可进入刷机模式
    没刷过的要短接，具体 bilibili 找个视频吧，文字不好讲清楚
  4. 连接成功后点击 usb_burn 的开始刷机即可
  5. 等待刷机完毕，先点击停止，关闭 usb_burn，看软件右下角操作说明第 6 条
  6. 拔掉所有线，接上网线以及电源线，等待开机完成，过程大概 3 分钟甚至更长时间，到路由器后台查看 dhcp 分配是否有 onecloud
  7. 复制 onecloud 的 ipv4 地址,并通过 ssh 连接

    输入账号 root
    密码 1234
    根据提示修改密码 就是再次输入两次自定义密码
    选择 bash 就是按 1
